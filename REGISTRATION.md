### Project Description (for registration form)

**Short Description (100 chars):**
Autonomous AI agent with self-custodial wallet that hunts bounties and settles value on-chain.

**Full Description:**
Atlas is an autonomous AI agent that operates as an independent economic entity. It hunts for bounty opportunities across GitHub, Upwork, and bug bounty platforms, evaluates ROI in real-time, executes tasks autonomously, and settles all value through self-custodial wallets using Tether's WDK.

**Key Features:**
1. Self-custodial agent wallet (WDK integration)
2. Autonomous bounty discovery and execution
3. Real-time economic tracking and ROI calculation
4. On-chain settlement and transparent ledger
5. Built-in safeguards (stop-loss, budget limits)

**Tech Stack:**
- OpenClaw agent framework
- Tether WDK (Wallet Development Kit)
- Python 3.12
- MCP (Model Context Protocol)
